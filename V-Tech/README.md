# V-Tech Innovations

A modern, minimalist website for V-Tech Innovations, showcasing our technology services and solutions.

## Features

- Responsive design
- Modern UI/UX
- Service showcase
- Contact form
- About company information

## Technologies Used

- React
- Tailwind CSS
- React Router
- React Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
